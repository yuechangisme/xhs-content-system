# 小红书合集轮转规则

本文档定义「懒人养生手册」后续发帖的合集轮转制。目标是让账号主页形成稳定栏目感，避免选题零散。

## 1. 合集名称

固定使用以下 5 个合集名：

1. 懒人饮食手册
2. 身体微习惯手册
3. 打工人恢复手册
4. 懒人运动手册
5. 好物避坑手册

## 2. 默认轮转节奏

采用 8 篇一个小循环：

1. 懒人饮食手册
2. 身体微习惯手册
3. 打工人恢复手册
4. 懒人运动手册
5. 懒人饮食手册
6. 身体微习惯手册
7. 打工人恢复手册
8. 好物避坑手册

好物类低频出现，避免账号显得像种草号。饮食、微习惯、恢复是主力内容，运动作为稳定补充。

## 3. 触发语

当用户说出以下类似表达时，默认进入下一篇帖子策划流程：

- 我们来发下一篇帖子
- 做下一篇
- 来一篇新的
- 下一篇小红书
- 今天发什么

Agent 不需要再询问要做哪个合集，应先根据轮转规则判断下一篇所属合集。

## 4. 判断下一篇合集的方法

触发下一篇帖子时，Agent 应按以下顺序判断：

1. 查看最近的正式内容目录，优先检查：
   - `content/投稿内容/待制作/`
   - `content/投稿内容/待投递/`
   - `content/投稿内容/已投递/`
2. 根据目录名、manifest、文案标题或用户最近确认的内容，判断上一篇所属合集。
3. 按 8 篇循环给出下一篇合集。
4. 如果无法可靠判断上一篇所属合集，说明不确定，并默认从「懒人饮食手册」开始新一轮。

不得手动修改 `state.json` 来记录或伪造轮转状态。

## 5. 自动提示格式

当用户触发下一篇帖子时，Agent 应先给出这样的提示：

```text
按合集轮转，这一篇轮到：<合集名>

这个合集适合做：<一句定位>
我建议这次从以下方向里选：
1. <选题方向 A>
2. <选题方向 B>
3. <选题方向 C>

视觉上会继续使用极简文字手册风格：大字封面 + 主题生活物件手账插图 + 轻分隔线内容页。
```

如果用户直接给出明确选题，则无需再发散太多方向，应只确认它归属哪个合集，并继续制作。

## 6. 各合集选题边界

### 懒人饮食手册

定位：吃什么、怎么吃、怎么少踩坑。

适合主题：

- 早餐、晚饭、外卖、便利店
- 甜饮、咖啡、奶茶、宵夜
- 胃胀、上火、熬夜后吃法
- 一周饮食小安排

### 身体微习惯手册

定位：不用大改生活的小习惯。

适合主题：

- 头油、久坐、喝水、洗澡、护眼
- 睡前收尾、午睡醒来、通勤小动作
- 枕套、梳子、坐姿、洗头水温等生活细节

### 打工人恢复手册

定位：疲惫、压力、熬夜、低能量后的恢复。

适合主题：

- 熬夜补救、下午犯困
- 压力肚、睡不踏实
- 周一低能量、加班后恢复
- 情绪内耗后的身体照顾

### 懒人运动手册

定位：不折腾、低门槛、能开始的动作。

适合主题：

- 散步、靠墙站、肩颈放松
- 饭后走、床上拉伸
- 久坐后的 3 分钟活动
- 不想运动的人先做什么

### 好物避坑手册

定位：低频出现，只做克制、实用、不强种草的好物内容。

适合主题：

- 保温杯、眼罩、枕头、梳子、坐垫
- 泡脚桶、热敷包、台灯
- 买前判断、低价替代、闲置避坑

写法要求：

- 不写买了就会变好。
- 必须说明什么人需要、什么人不需要。
- 必须给出不用买的替代方案或低成本做法。
